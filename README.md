UVic Cycle
========================

An app showcasing the possible features for UVic Cycle.
<br />

To download the app, click [here](https://github.com/david-ok/UVicCycle/raw/master/app/release/app-release.apk) (You must have "Unknown sources" enabled on your Android device, to learn how to click [here](https://www.applivery.com/docs/troubleshooting/android-unknown-sources))
<br />
